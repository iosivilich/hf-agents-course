## Contenido por unidad

### Unit 0 — Introduction to Agents
Configuración del entorno, conceptos fundamentales de agentes de IA: qué son, cómo funcionan, diferencia entre LLMs y agentes.

### Unit 1 — Build Agents with smolagents
Construcción de agentes con la librería `smolagents` de Hugging Face. Incluye el despliegue del agente en un Space de HF con búsqueda web y generación de imágenes.

**Space desplegado:** https://huggingface.co/spaces/iosiv/First_agent_template

### Unit 2 — Agent Frameworks
Uso de frameworks populares para construcción de agentes:
- **LangGraph**: agentes con grafos de estados
- **LlamaIndex**: agentes con indexación de documentos
- **LangChain**: cadenas de herramientas y agentes

### Unit 3 — Use Cases & Advanced Topics
Aplicaciones prácticas, agentes multimodales, benchmarks y evaluación de agentes.
